Options

This list includes all the options for tapShow.js.

# target

Type: String
Default: null
Description: Target option is to appoint which `container` to be initialized with tapShow.js plugin. It accepts CSS Selector as its input. 

                      
    tapShow({
        target: '.tpContainer'
    });

Properties Options for Container

# containerWidth

Type: String
Default: '100%'
Description: Define the width of container that holds all blocks. It supports length unit used in CSS.

# direction

Type: String
Default: 'horizontal'
Accept Values: 'horizontal' or 'vertical'
Description: This option uses flexbox direction approach. Basically, it helps you to set your block in horizontal arrangement or vertical arrangement.

# blockPosition

Type: String
Default: 'center'
Accept Values: 'center', 'top', 'bottom', 'left', or 'right'
Description: This helps you to set your block's position within container.

# textAlign

Type: String
Default: 'center'
Accept Values: 'left', 'center' or 'right'
Description: This apply the alignment of overall text in the block. To style individually between the blocks, or the content inside, please use options customCSS instead.

Properties Options for Block

# blockSelector

Type: String
Default: '.block'
Description: Developers can define the selector of the block(s). Be it in a class selector or ID selector.

# outerContentSelector

Type: String
Default: '.outer-content'
Description: Developers can define the selector of the outer-content. Be it in a class selector or ID selector.

# innerContentSelector

Type: String
Default: '.inner-content'
Description: Developers can define the selector of the inner-content. Be it in a class selector or ID selector.

# displayOne

Type: Boolean
Default: false
Description: Display one block at a time. If set to true, only one block will remain visible when user click on the block.

# allowCloseBtn

Type: Boolean
Default: true
Description: To enable the close button on top-right corner of each block.

# animateDuration

Type: Number
Default: 1200
Description: Duration of animation for the block to expand or collapse. Unit is measured in milliseconds (ms).

# fadeDuration

Type: Number
Default: 250
Description: Duration of fading for the content to become visible or invisible. Unit is measured in milliseconds (ms).

# expandWidth

Type: String
Default: '100%'
Description: Define the width of block during expansion. It supports length unit used in CSS.

# expandHeight

Type: String
Default: 'auto'
Description: Define the height of block during expansion. It supports length unit used in CSS.

# contentAlignH

Type: String
Default: 'center'
Accept Value: 'center', 'left' or 'right'
Description: To define the horizontal alignment for content within the block.

# contentAlignV

Type: String
Default: 'center'
Accept Value: 'center', 'top' or 'bottom'
Description: To define the vertical alignment for content within the block.

Basic CSS Style Options for Block

There are few CSS style options available. However, these styling can actually be overwritten in customCSS option. The reason for these few stylings were include originally is to provide simple styling to new users out of the box (example can be seem in demo page). For advance CSS styling, please use customCSS instead.

# blockWidth

Type: String
Default: '250px'
Description: CSS styling for block's width. It supports length unit used in CSS.

# blockHeight

Type: String
Default: 'auto'
Description: CSS styling for block's height. It supports length unit used in CSS.

# blockPadding

Type: String
Default: '20px'
Description: CSS styling for block's padding. This applied on four direction (top, bottom, left, right). It supports length unit used in CSS.

# blockMargin

Type: String
Default: '10px'
Description: CSS styling for block's margin. This applied on four direction (top, bottom, left, right). It supports length unit used in CSS.

# background

Type: String
Default: '#fff'
Description: Styling for block's background. It uses CSS background format.

# borderWidth

Type: String
Default: '1px'
Description: Styling for block's border width. It supports length unit used in CSS.

# borderColor

Type: String
Default: '#000'
Description: Styling for block's border color.

Events

There are few events occured in tapShow's blocks. Developers can insert a callback function in these events.

# showStart

Type: function
Description: When user click the box, it is the moment the inner-content is about to show.

# showEnd

Type: function
Description: When user click the box, it is the moment after the inner-content is shown.

# hideStart

Type: function
Description: When user close the box, it is the moment when the inner-content is about to hide.

# hideEnd

Type: function
Description: When user close the box, it is the moment after the inner-content is hide.

# blockExpand

Type: function
Description: When user click the box, it is the moment when the block expanded.

# blockCollapse

Type: function
Description: When user close the box, it is the moment when the block collapsed.

Styling

This is the advance customization for CSS Blocks.

# customCSS

Type: Object
Default: null
Description: Allow developers to have full control customization for how each blocks appearance.

Methods

With methods, developers able to control tapShow.js blocks externally. For Example, by incorporating the collapseBlock() method to an external button, user can collapse that particular blocks by click that external button.

# build()

Description: Allow developer to build tapShow.js blocks. This is launch initially when tapShow.js plugin is called.

# destroy()

Description: This method destroy the construction of tapShow's blocks.

# rebuild()

Description: Method to reconstruct the blocks.

# alignLeft()

Description: Method to cause all blocks align to left side within container.

# alignCenter()

Description: Method to cause all blocks align center within container.

# alignRight()

Description: Method to cause all blocks align to right side within container.

# expandBlock()

Description: Method to cause block to expand. If no selector is placed within, it will expand all blocks by default. This method accepts Selector. If a specific selector is placed within, it will expand that specific block only.

# collapseBlock()

Description: Method to cause block to collapse. If no selector is placed within, it will collapse all blocks by default. This method accepts Selector. If a specific selector is placed within, it will expand that specific block only.

# addBlock()

Description: Allow developers to add new block externally.

# removeBlock()

Description: Allow developers to remove block externally. This method accepts Selector. If a specific selector is placed within, it will remove that specific block only. If no argument is placed within, it will remove the last block by default.
