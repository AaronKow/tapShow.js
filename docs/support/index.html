Contributing

If you want to support tapShow.js with your amazing ideas or want to fix some bugs, your contributions to the project are always welcome, just go to tapShow.js Github and fork the project, then you can submit a Pull Request. :) Please make sure you follow these points:

  • Use the dev branch to commit your pull requests.
  • If the pull request is related with an issue, please name the issue number in the comments (eg: #24)

Appreciate much!

Contact

If you looking for help or wanted to ask a simple question, you can email to dxe.design.mail@gmail.com. I will try my best to asnswer all questions.

Report bug

Please report any bugs in tapShow.js's Github Issues page. If possible, please include a demo jsfiddle/codepen for reproducing that issue.
